# Cairo language common

This package provides shared, reusable functionalities for projects and packages that interact directly with the [Cairo](https://github.com/starkware-libs/cairo/) compiler.

## Support
The [CairoLS](https://github.com/software-mansion/cairols) and [Cairo lint](https://github.com/software-mansion/cairo-lint) teams are responsible for maintaining the repository, so for any questions you can reach us out on [Telegram](https://t.me/cairo_ls_support).
